# CoRF-Yoga

This repository contains the implementation of our IEEE Access paper:

"CoRF-Yoga: Cross-Modal Contrastive Learning for Robust Yoga Pose Recognition"

## Status
This repository is currently under active development. The full training and evaluation pipeline will be released upon acceptance.

## Structure
- pretrain/: contrastive learning modules
- downstream/: evaluation pipelines

## Dataset
The DSZ-Yoga dataset used in this study will be made publicly available on Zenodo:
https://zenodo.org/records/18297451

Please refer to the paper for details on data collection and preprocessing.

## Reproducibility
To support reproducibility, the implementation code is being actively prepared and will be fully released upon acceptance.
